Basic client for [Project Gemini](https://gemini.circumlunar.space/)
==

```
usage: explore.py [-h] [-p PORT] [-v] uri

positional arguments:
  uri                   gemini:// URI

optional arguments:
  -h, --help            show this help message and exit
  -p PORT, --port PORT  Override default port 1965
  -v, --verbose
```
